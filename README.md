# Dutch AI Tutor

A simple Next.js + TypeScript MVP for adults improving Dutch toward professional, real-life fluency.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000 to view the dashboard.

## MVP pages

- Dashboard: today's practice, weekly progress, mistake revisit cards, and sentence of the day.
- Practice session: realistic workplace prompt, vocabulary, sentence textarea, and mock AI feedback.
- Mistake bank: original sentences, corrections, explanations, tags, and revisit status.
- Progress: friendly analytics and repeated mistake patterns.

OpenAI and Supabase are intentionally not connected yet; all content uses mock data.
